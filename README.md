# H3MEX

Supports HotA v1.8.0.

This repository contains the GUI-focused version of H3MEX for Heroes 3: Horn of the Abyss map work.

## Launching

Run from the repository root:

`python h3mex.py`

If needed, install the GUI dependency first:

`pip install wxPython`

## Notes

- This repository is GUI-first.
- Terminal UI and legacy CLI flows are intentionally not included here.
